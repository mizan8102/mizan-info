# Assets Directory

## Profile Photo Instructions

To add your profile photo to the portfolio:

1. Save your professional photo as `mizan-photo.jpg` in this directory (`public/assets/`)
2. Recommended specifications:
   - Format: JPG or PNG
   - Size: 400x400 pixels (square aspect ratio)
   - File size: Under 500KB for optimal loading
   - Professional headshot with good lighting
   - Clear background (preferably solid color or professional setting)

## Current Photo Usage

The photo is used in:
- Cover page (200x200px circular crop)
- About page header
- Contact page
- All other portfolio sections
